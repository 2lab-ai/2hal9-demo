API Reference

Create a New Game

POST /api/v1/games 
{
  "game_type": "PrisonersDilemma",
  "players": ["player1", "player2"],
  "config": {
    "rounds": 100,
    "time_limit_ms": 5000
  }
}

Get Game Status

GET /api/v1/games/{game_id}

Submit Action

POST /api/v1/games/{game_id}/actions 
{
  "player_id": "player1",
  "action": "cooperate"
}
